かんたん！GitHubでウェブサイトを公開しよう

小学生でもわかる！ステップ・バイ・ステップガイド

準備するもの

  • • GitHubアカウント

    無料で作れるよ！メールアドレスが必要だね

  • • 公開したいウェブサイトのファイル

    index.htmlファイルを用意しよう

かんたん3ステップ

1 リポジトリを作る

  1. GitHubにログインする
  2. 右上の「+」ボタン → 「New repository」を選ぶ
  3. リポジトリ名を「ユーザー名.github.io」にする
  4. 「Public」を選んで「Create repository」をクリック

2 ファイルをアップロード

  1. 「Add file」→「Upload files」を選ぶ
  2. ウェブサイトのファイルをドラッグ＆ドロップ
  3. 「Commit changes」をクリック

3 公開設定をする

  1. 「Settings」タブを開く
  2. 左メニューの「Pages」を選ぶ
  3. Source を「main」に設定
  4. 数分待つと公開完了！

できあがり！

これで完成！「https://ユーザー名.github.io」にアクセスすると、 自分のウェブサイトが見られるようになるよ！

※反映には数分かかることがあるから、少し待ってね。
